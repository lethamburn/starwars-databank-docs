# Star Wars Databank Docs

Documentation site for the Star Wars Databank API, built with Next.js and Nextra.

## Stack

- Next.js
- Nextra
- Nextra Docs Theme

## Requirements

- Node.js 18
- npm 10

## Development

```bash
npm install
npm run dev
```

The site will be available at `http://localhost:3000`.

## Production Build

```bash
npm run build
npm run start
```

## API Base URL

`https://starwars-databank-server.onrender.com/api/v1/`

## Notes

- The project is configured to use `npm`.
- `pnpm-lock.yaml` has been removed to avoid inconsistent installs in deployment environments.
