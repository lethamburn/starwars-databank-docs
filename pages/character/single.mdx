# Get a single character

## Get by a id

You can get a single character by adding the id as a parameter: /characters/id

```text /GET/
GET https://starwars-databank-server.vercel.app/api/v1/characters/640b2d19c916010a8aa5a88e
```

## Filter by name

You can get some characters by adding the name as a parameter with the added endpoint name: /characters/name/CharacterName

```text /GET/ /Plo%20Koon/
GET https://starwars-databank-server.vercel.app/api/v1/characters/name/Plo%20Koon
```
